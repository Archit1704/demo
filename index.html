What This Site Includes

Static Files

No build process is required. Deploy the files directly with SCP, rsync, or your preferred release step.

Responsive Layout

The page works on desktop and mobile, which makes it easy to verify after each deployment.

Visible Version

Update the release label below whenever you want to confirm that a new deployment reached your server.

Workflow

Suggested CI/CD Flow

  1. Push changes to your GitHub repository.
  2. GitHub Actions runs on the selected branch.
  3. The workflow copies these static files to your EC2 web directory.
  4. Your EC2 web server serves the updated page.
